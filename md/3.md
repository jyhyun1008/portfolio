- 미스키의 오리지널 스크립트 언어 ‘아이스크립트’ 의 개변, ‘파이스크립트’ 로 리브랜딩

- 커스텀 이모지 관리를 쉽게 하는 기능
- 간단한 CSS 수정(폰트 수정, 마우스커서 수정, UI의 수정)
